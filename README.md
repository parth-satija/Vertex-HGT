Vertex-HGT (Heterogenous Geometry Terrain) is a high-performance, GPU-driven heightmap-based terrain rendering engine designed to push the limits of modern hardware by intelligently distributing rendering workloads between integrated and dedicated GPUs. Moving away from traditional, CPU-heavy quadtree mesh optimization, this project implements a massive-parallel chunking and Level of Detail (LOD) system directly on the GPU using C++ and GLSL compute shaders to maximize framerates and minimize memory overhead. Please note: This project is actively in its development and research phase; the core architecture is highly experimental, and full documentation, benchmarking data, and build instructions will be finalized as the engine nears its initial stable release.
